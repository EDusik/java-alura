## :computer: Commands 
<div id="commands"> 

```eclipse ide

Fecha menus: `CRTL M`;
Criar nova classe: `CTRL N`;
Abre menu de pesquisa global: `CTRL 3`;
Cria variável: `CTRL 1`;
Roda a aplicação: `CTRL F11`;
Cria método main(): `main + CTRL + backspace`;

`F3` em cima da classe: Vai até a classe, o mesmo para métodos;
(Na Linha) CTRL + ALT + (Seta para baixo): Repete a linha para baixo;
(Na Linha) CTRL + ALT + (Seta para cima ): Repete a linha para cima;
Procurar arquivo: `CTRL + SHIFT + R`  (Podemos utilizar "*" se não sabemos o nome);
Procurar classes: `CTRL + SHIFT + T`;
Procurar métodos nas classes: `CTRL + 0`;
Ver subclasses de uma classe: `CTRL + T`;
Voltar ou ir para frente: ALT + setas <- ou ->;

Encontra onde os métodos estão sendo utilizados: `CTRL + SHITF + G`;
Encontra onde os métodos estão sendo utilizados e também seus filhos: `CTRL + SHITF + H`;
Busca textos dentro de um arquivo: `CTRL + F`;
Busca incremental: `CTRL + J`;
Busca Geral: `CTRL + H` (CTRL . = proximo resultado, CTRL , = resultado anterior);
Mostrar classes que utilizam determinado método: `CTRL + ALT + H`;
Fechar todas janelas abertas: `CTRL + SHIFT + W`;
Refactor: `ALT + SHIFT + T`;
Organiza os imports: `CTRL + SHIFT + O`;
Gerar delegate métodos: `CTRL + 3 e digite delegate` (também é utilizado para outras configs);

// Task tags
// TODO algo que precisa ser feito
// FIXME algo que precisa ser corrigido
// XXX algo que precisa ser refatorado

Inicia a aplicação web: `CTRL + ALT + R`;
Formatar arquivo: `CTRL SHIFT + F`;
Renomear: `ALT + SHIFT + R`;

```
