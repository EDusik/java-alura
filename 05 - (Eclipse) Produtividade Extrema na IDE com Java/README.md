## :computer: Commands 
<div id="commands"> 

```eclipse ide

// CRTL M: Fecha menus;
// CTRL N: Criar nova classe;
// CTRL 3: Abri menu de pesquisa global;
// CTRL 1: Cria variável;
// CTRL F11: Roda a aplicação;
// main + CTRL + backspace: Cria método main();
// F3 em cima da classe: Vai até a classe, o mesmo para métodos;
// (Na Linha) CTRL + ALT + (Seta para baixo): Repete a linha para baixo;
// (Na Linha) CTRL + ALT + (Seta para cima ): Repete a linha para cima;
```
